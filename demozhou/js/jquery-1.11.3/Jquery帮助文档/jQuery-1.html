Jquery简介

什么是jquery？

jQuery是一个快速、简洁的JavaScript库(工具箱)。

简单的说就是封装了JS中常用的功能代码，更便于我们的DOM操作。

如：获取页面上面的一个id为box的元素：

  • js原生: var box = document.getElementById('box');
  • jQuery方式: var box = $("#box");

jQuery设计的宗旨是"write Less，Do More"，即倡导写更少的代码，做更多的事情。

jQuery选择器

什么是选择器？

就是选择页面上元素的一种工具。

  • 类似于js中获取指定id元素，document.getElementById();
  • 类似于js中获取指定class元素，document.getElementsByClassName();
  • 类似于js中获取指定标签元素，document.getElementsByTagName();

上面是js元素的选择器(类似于css选择器)，换成jQuery方式如下：

  • $("#id")
  • $(".class")
  • $("div ")

在jQuery中选择器有很多用法：这里重点掌握以下几种选择器，开发中基本满足使用。

  • id选择器 $("#id")
  • class选择器 $(".class")
  • 层级选择器 $("parent > child") 或 $("parent childs")
  • 标签加属性组合选择器 $("ele[attr=value]")

更多用法可以参考jQuery手册:https://jqueryapi.net/

筛选选择器的使用

简单选择器就是对匹配到的元素进行筛选，使用起来非常简单。

语法:

  • :first 匹配第一个元素

  • :last 匹配最后一个元素

  • :even匹配下标值为偶数的所有元素

  • :odd 匹配下标值为奇数的所有元素

  • :eq(index) 匹配下标为指定值的元素

  • :gt(index) 匹配下标大于指定值的所有元素

  • :lt(index) 匹配下标小于指定值的所有元素

  • :not(selector) 匹配除了指定选择器以外所有元素

  • :contains(text) 匹配包含文本text的元素,类似模糊查询

使用范例：

  • 匹配第一个div元素：$('div:first')或 $('div:first-child')
  • 匹配最后第一个div元素：$('div:last') 或 $('div:last-child')
  • 匹配下标为偶数的div元素：$('div:even')
  • 匹配下标为奇数的div元素：$('div:odd')
  • 匹配下标为2的div元素：$('div:eq(2)')
  • 匹配下标大于2的div元素：$('div:gt(2)')
  • 匹配下标小于2的div元素：$('div:lt(2)')
  • 匹配除了id不为box的div元素：$('div:not(#box)')
  • 匹配内容含关键字蛋的div元素：$('div:contains(蛋)')

DOM对象和jquery对象

什么是DOM对象

就是通过原生的js中的方法来获取到的对象，如下：

  • document.getElementById()
  • document.getElementsByTagName()
  • document.getElementsByName()
  • document.getElementsByClassName()

获取到的对象都称之为DOM对象。

什么是jQuery对象

若通过jQuery选择器$("选择器")获取到的对象都称之为jQuery对象。

DOM对象和jQuery对象的关系

  • DOM对象是数组下标的形式存在jQuery对象中。

问题？

  • 问1：jQuery对象与DOM对象的方法是否能够相互的使用？
  • 问2：jQuery对象是否能够使用DOM对象的方法、属性、事件？
  • 问3：DOM对象能否使用jQuery对象的方法？

答:都是可以的。只要DOM对象和jQuery对之间进行相互的转换即可实现！

DOM对象和jquery对象互相转换

1、jquery对象转化为DOM对象

  • 方式一:（推荐）
 jquery对象[下标值]=>获取到对应下标的DOM对象
  • 方式二:
 jquery对象.get(下标值)=>获取到对应下标的DOM对象

2、DOM对象转化为jquery对象

把DOM对象用小括号括起来即可转化为jQuery对象

$(DOM对象)=>将DOM对象转为jQuery对象

问题：事件处理程序中的this是DOM对象还是jquery对象？

答：this是当前元素的DOM，$(this)才是jquery对象。

window.onload和$(function(){})的区别?

  • window.onload是在等待网页中所有的元素(img、js、image)完全加载到浏览器后才执行。
  • $(document).ready() 是在DOM结构载入就绪后开始执行。

这里推荐使用$(function(){ })形式;因为此方法速度快于window.onload=function(){}因为不需要加载外部的资源。

【重点】jQuery事件绑定

注：在jquery中事件都是不要加on前缀，原生js需要加on

1、通过事件名直接绑定。如：jQuery对象.事件名( function(){} )

2、在jq中可以使用bind方法进行事件的绑定：

  • bind(type,callback) ：对事件进行绑定 只能绑定一个事件

    参数说明：
    type：指事件名 但是这里的事件名不要带on前缀。

    callback：匿名函数事件的处理程序

3、在jq中可以使用on方法进行事件的绑定：

  • on(type,callback) 只能给页面已有的元素的绑定事件。等价于上面两种（事件名和bind方式 ）

    参数说明:

    type：事件名

    callback：事件的处理程序

  • on( type,selector,callback) 在已有的元素上绑定代理的事件。（事件委托的方式）

    参数说明：

    • type ：事件名
    • selector:子选择器
    • callback：事件的处理程序

    委托的调用形式：父选择器.on(type,子选择器,callback);
