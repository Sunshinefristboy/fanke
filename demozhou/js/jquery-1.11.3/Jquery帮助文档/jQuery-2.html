jQuery动画效果

滑动效果

  • slideDown()：将隐藏的元素显示出来

  • slideDown(speed,[callback]) ：以设定speed的速度将隐藏的元素显示出来

    其中speed这个参数有两种表示方式 ：毫秒数或英文单词如"slow", "normal", "fast"

  • slideUp()：将显示的元素隐藏

  • slideUp(speed,[callback]) ：以设定speed的速度将显示的元素隐藏

  • slideToggle()：在显示或者隐藏间滑动切换

  • slideToggle(speed,[callback]) ：以设定speed的速度显示或者隐藏

淡入淡出

  • fadeTo(time,opacity)：设定指定的时间实现淡入或淡出到指定的透明度

    ​ 参数说明：

    ​ time:毫秒值

    ​ opacity:透明度，取值为0至1之间，表示透明度的数字。

  • fadeIn()：淡入（显示）

  • fadeOut()：淡出（隐藏）

  • fadeToggle()：淡入或者淡出开关

自定义动画

语法：

animate(styles,[speed],[callback])

在规定的时间内完成动画的执行！

参数说明：

  • styles：要求是一个JSON格式对象,样式的值和css方法设置是一样的，其中样式值只有数字值才可创建动画（如：font-size:30px）,字符串值无法创建动画（如：background-color:red）。

  • speed：动画完成所需要的时间

  • callback：完成动画后执行的回调函数

延时执行动画delay

语法：

delay(time)

作用：设置一个延时时间time来执行队列中的动画。

参数：

  • time:延时时间，单位：毫秒

如：

$("button").click(function(){
    $("#div2").delay("fast").fadeIn();
    $("#div3").delay(800).fadeIn();
});

节点查找（链式调用）

链式编程(方法返回的都是对象可以连续调用)，即方法return this；

形如：对象.方法().方法().方法().......

形如：对象.属性.方法().

promise.then().then()

注：是否可以往后面链式调用，看前面的方法是否返回一个对象，是的话就可以调用其对象的相关方法或属性。

常用于节点查找的方法如下： 以下方法都只能是jQuery对象进行调用

  • eq(index) ：获取指定下标的元素

  • filter(expr) ： 将查找到的元素进行过滤筛选，查找符合筛选条件元素

  • not(expr) ：获取不符合某个匹配条件的元素

  • next([expr]) ：获取当前元素的下一个兄弟元素

  • prev([expr])：获取当前元素的上一个兄弟元素

  • parent([expr])：获取当前元素的父元素

  • parents([expr])：获取当前元素所有的祖先元素

  • siblings([expr]）：获取除了当前元素以外的所有兄弟元素

  • children([expr])：获取当前元素的==子元素==

  • find(expr):获取当前元素下满足条件的==子孙元素==,其中expr参数必写

  • last()：获取最后一个元素

  • first()：获取第一个元素

    其中参数expr写选择器。

链式调用注意的地方

注意：

1、 调用一个方法的时，此方法可能会涉及到对象的转移(即调用不同的方法会返回不同的对象

2、 给对象设置一些值，对象是不会发生转移，如通过以下方法进行设置还是返回当前的jQuery对象,则可以继续向后链式调用。
- attr('k’,'v')
- addClass('class1')
- removeClass('class1')
- css('k','v')
- html(html)
- ...

3、 调用某些方法如果不是返回jQuery对象，如返回一个字符串。则不可以继续向后链式调用。如：

- html()：获取标签的值
- attr('attr')：获取属性值。
- css('font-size'); 获取演示

jQuery中插件机制

介绍

我们去工作中发现我们需要实现某一个功能，需要用到一个方法 但是jQuery并没有给我们进行封装这个方法供jQuery对象使用，但jQuery它给我们预留了一个扩展方法的机制！

格式：

$.fn.extend(object)
或
jQuery.fn.extend(object)

其中参数object是个json格式对象。

注意：在jQuery中jQuery等价于$符号，他们是相等的

console.log( jQuery === $ ); // true
var ul = $("#ul"); 
// 等价于 
var ul = jQuery("#ul");

具体使用

语法：

 $.fn.extend({
         "方法名1":function(){},
	    "方法名2":function(){},
	    //定义更多..
 });

或：

 $.fn.方法名 = function(){
	// to do...
 }

注意：

​ 上面的方法，都是可以直接通过jQuery对象直接调用。

​ 在整个jQuery中除了扩展机制中的方法this是指向的当前jQuery对象 ，其它方法中的this都是指向当前DOM对象。

即使：

$.fn.extend({
	方法名:function(){ 
        // this是当前jQuery对象 
    },
});

//调用
$('ul').方法名();

 }

jQuery中的Ajax操作

介绍

在jQuery中发送一个Ajax请求常用的有以下几种

  • $.ajax()
  • $.get()
  • $.post()

①方式一 ： $.ajax( {} )

此方法功能最强大，get和post请求都可以发送，回调函数也最多，如：发送之前、发送失败、发送成功、发送完毕的回调函数都可以设置。

参数说明：

object：是个json格式的对象，常用的键值有以下几种

键名说明
type请求的方式。常用的有get和post,默认为get请求
url请求的地址。
data携带的请求参数，格式为请求字符串id=3&age=28或者json格式 {id:3,age:28}
async是否异步请求，默认为true异步，false同步
dataType指定服务器返回的数据类型，值有text、json、xml等
cache是否允许缓存，默认为true缓存，false不缓存
success请求成功的回调函数，相当于原生Ajax请求状态等于4
beforeSend发送请求之前触发的回调函数，相当于原生Ajax请求状态不等于4,一般用做loading提示加载等效果。
complete请求完毕的回调函数，无论失败或成功都会触发
error发送失败的回调函数，如请求的文件不存在会触发

②方式二：简单易用的高级实现 （底层也是使用$.ajax实现的）

语法格式：

$.get(url, [data], [callback], [type])

$.post(url, [data], [callback], [type])

说明：.get()只能发送get请求，.post()只能发送post请求，且这两者都是异步请求。

参数说明：

url：请求的地址

data：请求的参数，请求字符串或者json格式

callback：请求成功的回调函数 相当于原生Ajax请求状态等于4

type：指定服务端返回的数据类型,有xml, json, text等
